# 🧪 Лабораторные работы

## Lab 2 — Проверка принадлежности точки области

**Задание**

Ввести вещественные числа **X** и **Y** и определить принадлежит ли точка с координатами **(X,Y)** заштрихованной части плоскости.

Необходимо протестировать **все ветви алгоритма**.

**Навыки**

- условные операторы  
- логические выражения  
- тестирование ветвлений  

---

## Lab 3 — Работа с циклами

**Задание**

В заданном целом числе определить **количество нечетных цифр**.

Ограничения:

- нельзя использовать строки  
- нельзя использовать массивы  

Задача должна быть решена с использованием:

- `for`
- `while`
- `do while`

После анализа необходимо выбрать **наиболее рациональный цикл**.

---

## Lab 4 — Геометрия и расстояния

Дана прямая:
y = kx + b

где **k** и **b** вводятся с клавиатуры (k ≠ 0).

Далее вводятся координаты **20 точек по оси OX** на этой прямой.

Необходимо:

- найти пару точек с **минимальным расстоянием**
- вывести координаты всех точек
- вывести **номера найденной пары**
- вывести координаты найденных точек

---

## Lab 5 — Работа с матрицами

Дан массив целых чисел **3 × 12**.

Массив формируется с помощью генератора случайных чисел в диапазоне:
[-20 ; 70]

Необходимо:

- определить **среднее арифметическое отрицательных элементов**

Программа должна вывести:

- исходную матрицу  
- количество отрицательных элементов  
- их среднее арифметическое  

---

## Lab 6 — Функции и процедуры

Дана матрица **A(n, m)**, где:
n, m ≤ 10

Значения элементов находятся в диапазоне:
[-10 ; 10]

Задача:

1. Найти **минимальный элемент каждой строки**
2. В зависимости от его знака выполнить **сдвиг строки**

- вправо  
- влево  

Количество сдвигов равно **модулю минимального элемента**.

Программа должна использовать **6 функций**.

Вывести:

- исходную матрицу  
- столбец минимальных элементов  
- результирующую матрицу после сдвигов  

---

## Lab 7 — Обработка строк

Дана последовательность слов из **строчных латинских букв**.

Формат:
word, word, word.

Условия:

- количество слов больше 5  
- между словами стоит запятая и пробел  
- после последнего слова стоит точка  

Необходимо:

- определить количество слов, которые начинаются **с гласной буквы**
- вывести эти слова

---

## Lab 8 — Связные списки

С клавиатуры вводятся целые числа.

Необходимо:

1. создать список с **удалением повторяющихся чисел**
2. вывести полученный список
3. удалить **максимальный элемент**
4. вывести удаленное число
5. вывести новый список

Создать **бинарный файл F**, состоящий из целых чисел.

Необходимо:

переписать из файла **F** в файл **G** все **нечетные числа**.

## Lab 10 — Исследование погрешностей вычислений

Цель работы — исследовать **погрешности представления чисел в памяти компьютера**.

Исследуются типы данных:
float
double
long double

Необходимо:

- оценить погрешность представления дроби **1/3**
- вычислить **абсолютную и относительную погрешность**
- исследовать влияние типа данных на точность вычислений

Также вычисляется выражение:
ch²x − sh²x = 1

и проводится анализ точности вычислений.

## Lab 11 — Динамические массивы и указатели

Создать динамический массив:
A(n)

где **n вводится с клавиатуры**.

Необходимо:

- найти **сумму отрицательных элементов массива**
- записать эту сумму **в последний элемент массива**
- вывести **адреса отрицательных элементов через указатели**

Вывести:

- исходный массив  
- измененный массив  
- найденную сумму

## Lab 12 — Работа с динамическими матрицами

### Часть A

Создать матрицу:
(N+1) × N

Необходимо:

- удалить **наибольшие элементы каждого столбца**
- получить квадратную матрицу **N × N**

Вывести:

- исходную матрицу  
- полученную матрицу  
- адреса найденных максимальных элементов  

---

### Часть B

Дана квадратная матрица **размером не более 10 × 10**.

Необходимо вывести матрицу **в виде параллелограмма**.

## Lab 13 — Работа с текстом

С клавиатуры вводится текст, содержащий **более 10 слов различной длины**.

Необходимо:

- распечатать текст по строкам
- строка должна содержать **12 символов**
- если встречается пробел — строка заканчивается

Дополнительно:

- вывести **адреса букв третьего слова** с использованием указателей

# 🏠 Домашние задания

## Homework 1

### Часть 1 — Вычисление выражений

Вычислить заданное выражение и определить **влияние выбранного типа данных на точность вычислений**.

---

### Часть 2 — Разветвляющийся процесс

Заданы три действительных числа.

Необходимо присвоить логической переменной значение **true**, если **только одно из чисел положительное**, иначе **false**.

Программа должна протестировать **все ветви алгоритма**.

---

### Часть 3 — Циклические вычисления

Вычислить значение функции **в точке x** с заданной точностью **ε**.

Необходимо определить:

- как меняется число итераций
- как влияет точность вычислений

Результаты сравнить с точным значением, полученным стандартной функцией.

---

## Homework 2

### Часть 1 — Сортировка массива

Отсортировать одномерный массив вещественных чисел **по невозрастанию**.

Метод сортировки:
Bubble Sort (метод обменов)

---

### Часть 2 — Работа с матрицами

В матрице необходимо:

- найти строку с **первым отрицательным элементом**
- найти строку с **максимальным положительным элементом**
- поменять их местами

Вывести:

- исходную матрицу  
- полученную матрицу  
- координаты найденных элементов  

Если строки совпадают — вывести диагностическое сообщение.

---

## Homework 3

### Часть 1 — Модули и указатель на функцию

Разработать модуль, содержащий процедуру вычисления **площади под графиком функции**.

Используется формула численного интегрирования.

В основной программе проверить процедуру на функциях:
y = sin(x)
y = x²

Результаты сравнить с точным значением.

---

### Часть 2 — Строки и динамические структуры

Дана строка слов, разделенных пробелами.

Необходимо:

- сформировать **однонаправленный список**
- определить количество слов длиной меньше **N**
- вывести сформированный список и результат поиска

Если подходящих слов нет — вывести сообщение.
